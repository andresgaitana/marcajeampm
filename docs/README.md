Manual entregable por nivel de usuario:

### Cómo convertirlos a PDF (para entregar impreso)
Cualquiera de estas opciones:
1. **Word / Google Docs:** abre el `.md`, copia el contenido, pégalo en un documento y exporta a PDF (Archivo → Descargar/Guardar como PDF).
2. **VS Code:** instala la extensión *"Markdown PDF"*, abre el `.md`, clic derecho → *Export (pdf)*.
3. **Navegador:** ábrelo con cualquier visor de Markdown y usa *Imprimir → Guardar como PDF*.

> ¿Quieres que te los genere ya en **PDF con el logo de AM/PM** y formato de marca? Avísame y te los preparo.
